# jimwoodswrites.com — Project Brief

## Overview
Personal author and book publishing consultant website for Jim Woods.
Domain: jimwoodswrites.com | Hosting: Vercel | Repo: https://github.com/Jimwoods135/jimwoodswrites

## Tech Stack
- **Framework:** Next.js (App Router) + TypeScript
- **Styling:** Tailwind CSS
- **Hosting:** Vercel
- **CMS:** TBD — static for Phase 1; evaluate Sanity for Phase 2+ if Jim needs self-serve content updates

## About Jim
- Local author and book publishing consultant based in **Cuyahoga Falls, Ohio**
- Tagline: **"Let's bring your book into the world"**
- Niche: Non-fiction (business, self-help, memoir with application); also consults on fiction
- **9 published books:**
  - 2 business/productivity
  - 4 westerns
  - 3 crime novels
  - Book #10 (upcoming): **Story Builder** — showcases his framework, directly tied to consulting brand
- **Podcast:** Story Builder — on YouTube and Spotify

## Services
1. Book Coaching
2. Developmental Editing
3. Book Formatting
4. Self-Publishing Services
5. Book Consulting

## External Links & Integrations
- Amazon author page (all 9 books)
- LinkedIn (serves as About/bio)
- Email contact: mailto link
- Podcast: YouTube + Spotify
- Booking: TidyCal link
- Newsletter: Substack (existing list)

## Design Direction
- **Palette:** Dark blues, blacks, white — clean and minimal
- **Aesthetic references:** mikevardy.com (calm, editorial, personal authority) + copyblogger.com (clear service hierarchy, conversion-focused)
- **Typography:** Literary authority feel — serif headings, clean sans-serif body
- **Assets available:** Headshot, book cover images, stock lifestyle photos
- **Logo:** None — use typographic wordmark

## Target Audience
- Aspiring non-fiction authors (business, self-help, memoir)
- Entrepreneurs and professionals who want to write a book
- Local clients in Cuyahoga Falls / Summit County / greater Akron, Ohio area

## Primary CTA Goal
Connect potential clients with Jim — via TidyCal booking or email contact.

## Phases

### Phase 1 — Design System & Core Site Build
- Initialize Next.js + Tailwind project
- Define design tokens: color palette, typography, spacing scale
- Create typographic wordmark (no logo)
- Build all pages with placeholder content and images
- Sections: Hero, Services, Books, About snippet, Podcast, Testimonials, Newsletter (Substack), Footer
- All integrations wired: TidyCal, Substack, Amazon, LinkedIn, YouTube/Spotify, mailto
- Fully responsive (mobile-first)

### Phase 2 — Content Integration
- Swap in real headshot and book cover images
- Final copy from Jim
- Live testimonials
- Story Builder (book #10) launch-ready featured section
- Evaluate CMS if Jim needs self-serve updates

### Phase 3 — Technical SEO Foundation
- Meta tags, Open Graph, Twitter cards on all pages
- JSON-LD structured data: Person, LocalBusiness, Book schemas
- sitemap.xml + robots.txt
- Core Web Vitals optimization
- Wire up Google Analytics + Search Console

### Phase 4 — Local SEO
- Google Business Profile audit and optimization
- Local keyword strategy: Cuyahoga Falls, Summit County, Akron area
- Local-targeted landing page content
- NAP (Name/Address/Phone) consistency across all platforms
- Local citation building

### Phase 5 — Content & Growth Strategy
- Blog/article content calendar targeting primary keywords
- Story Builder launch content integration
- Podcast SEO (YouTube + Spotify discoverability)
- Substack cross-promotion strategy
- Long-term keyword and authority building plan
